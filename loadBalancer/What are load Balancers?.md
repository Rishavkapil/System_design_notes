
Load balancers distribute incomming client requests to compute resources such as application servers and databases. 

In each case, the load balancer returns a response from the computing resources to the appropriate client. Load Balancers are effective at : 

* Preventing request from going to unhealthy servers. 
* Preventing overloading resources. 
* Helping to eliminate a single point of failure.

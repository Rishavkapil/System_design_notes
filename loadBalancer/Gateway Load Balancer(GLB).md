
### How a Gateway load Balancer works 

With GLB, you can deploy, manage, and scale virtual appliances, such as intrusion detection and prevention, firewalls, deep packet inspection systems. 

It creates a single entry and exit point for all application traffic 


It is mainly used for Security tools like : 

* Firewalls
* Intrusion detection system
* Intrusion Prevention System
* Packet inspection system
Basically, tools that inspect or filter every packet before it reaches your application.
